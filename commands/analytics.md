# Analytics Command

Compute agent metrics, session stats, event distribution, and delegation graphs from work tracker data.

See `skills/analytics/SKILL.md` for full documentation.

## Usage

```
/team-shinchan:analytics
/team-shinchan:analytics --report full
/team-shinchan:analytics --agent bo
/team-shinchan:analytics --trace trace-1709312400000-a3f1
/team-shinchan:analytics --period 7d
```
