# Budget Command

View current budget status or configure turn limits for the active workflow.

See `skills/budget/SKILL.md` for full documentation.

## Usage

```
/team-shinchan:budget
/team-shinchan:budget --set total=200
/team-shinchan:budget --set phase=50
/team-shinchan:budget --reset
/team-shinchan:budget --reset phase
```
