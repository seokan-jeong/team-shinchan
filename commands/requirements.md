# Requirements Command

Analyzes a feature or task to uncover hidden requirements, edge cases, risks, and unstated expectations.

See `skills/requirements/SKILL.md` for full documentation.

## Usage

```
/team-shinchan:requirements [feature or task to analyze]
```

## Examples

```
/team-shinchan:requirements "Add user authentication to the app"
/team-shinchan:requirements "Build a file upload feature"
/team-shinchan:requirements "Implement rate limiting on the API"
```
