# Research Command

Invokes Masumi for web research, documentation lookup, and knowledge gathering.

## Usage

```
/team-shinchan:research "topic or question"
```

## Examples

```
/team-shinchan:research "Best practices for JWT refresh token rotation"
/team-shinchan:research "Compare Prisma vs Drizzle ORM"
/team-shinchan:research "React Server Components architecture"
```
