# Test-Driven Development Command

Enforces the RED-GREEN-REFACTOR cycle for all new code and bug fixes during Stage 3 execution.

See `skills/test-driven-development/SKILL.md` for full documentation.

## Usage

```
/team-shinchan:test-driven-development [task description]
```

## Examples

```
/team-shinchan:test-driven-development "Add date parser utility"
/team-shinchan:test-driven-development "Fix off-by-one error in pagination"
/team-shinchan:test-driven-development "Refactor auth token validation"
```
