# Verification Before Completion Command

Ensures no task is marked complete without actual test/build evidence.

See `skills/verification-before-completion/SKILL.md` for full documentation.

## Usage

```
/team-shinchan:verification-before-completion
```

## Examples

```
/team-shinchan:verification-before-completion
```

This skill is typically invoked automatically by agents before reporting task completion. It is not intended for direct user invocation.
