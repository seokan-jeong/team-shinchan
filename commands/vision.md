# Vision Command

Visual content analysis including screenshots, UI mockups, architecture diagrams, and PDF documents.

See `skills/vision/SKILL.md` for full documentation.

## Usage

```
/team-shinchan:vision [file path or description of visual content]
```

## Examples

```
/team-shinchan:vision "screenshot.png - analyze the UI layout"
/team-shinchan:vision "design-mockup.pdf - extract component specifications"
/team-shinchan:vision "architecture-diagram.png - describe the system design"
```
