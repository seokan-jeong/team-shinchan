# Autopilot Skill

자율 실행 모드를 활성화하여 사용자 개입 없이 작업을 완료합니다.

## 트리거

- "autopilot"
- "자동으로", "알아서", "auto"

## 활성화 시 동작

1. **요구사항 분석**: 봉미선(Metis)이 요구사항 분석
2. **계획 수립**: 유리(Planner)가 작업 계획 수립
3. **작업 분배**: 짱구(Orchestrator)가 에이전트들에게 작업 배분
4. **병렬 실행**: 여러 에이전트가 동시에 작업 수행
5. **품질 검증**: 액션가면(Reviewer)이 결과 검증
6. **자동 수정**: 문제 발견 시 자동으로 수정

## 에이전트 협업

```
봉미선 (분석)
    ↓
유리 (계획)
    ↓
짱구 (오케스트레이션)
    ↓
┌───┬───┬───┐
맹구  수지  흑곰 (실행)
└───┴───┴───┘
    ↓
액션가면 (검증)
```

## 완료 알림

모든 작업이 완료되면:
- 완료된 작업 목록 표시
- 변경된 파일 목록 표시
- 다음 단계 제안
