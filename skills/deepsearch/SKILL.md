# Deepsearch Skill

흰둥이(Explorer)와 채성아(Librarian)가 코드베이스를 깊이 탐색합니다.

## 트리거

- "deepsearch", "깊은검색"
- "찾아줘", "search"

## 탐색 방법

### 1단계: 빠른 탐색 (흰둥이)
- 파일명 패턴 매칭
- 키워드 검색
- 디렉토리 구조 파악

### 2단계: 심층 탐색 (채성아)
- 코드 내용 분석
- 관련 문서 검색
- 의존성 추적

## 검색 대상

| 대상 | 설명 |
|------|------|
| 파일 | 파일명, 경로로 검색 |
| 함수 | 함수명, 시그니처로 검색 |
| 클래스 | 클래스명, 상속 관계로 검색 |
| 문서 | README, 주석, 문서 검색 |
| 패턴 | 코드 패턴으로 검색 |

## 결과 형식

검색 결과 제공:
- 파일 경로 및 라인 번호
- 관련 코드 스니펫
- 컨텍스트 설명
- 관련 파일 목록
