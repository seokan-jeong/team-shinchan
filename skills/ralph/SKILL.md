# Ralph Skill

작업이 완전히 완료될 때까지 절대 멈추지 않는 끈기 모드입니다.

## 트리거

- "ralph"
- "끝까지", "완료할 때까지"
- "dont stop", "don't stop"

## 활성화 시 동작

1. **무한 반복**: 작업이 완료될 때까지 계속 시도
2. **자동 복구**: 에러 발생 시 자동으로 해결 시도
3. **진행 추적**: TODO 리스트로 진행 상황 추적
4. **검증 필수**: 액션가면(Reviewer)의 최종 검증

## Ralph 루프

```
1. TODO 리스트 확인
2. 다음 작업 수행
3. 결과 검증
4. 실패 시 → 원인 분석 → 재시도
5. 성공 시 → 다음 작업으로
6. 모든 작업 완료 → 최종 검증
7. 검증 실패 → 수정 후 재검증
```

## 완료 조건

다음 모든 조건 충족 시에만 완료:
- TODO 리스트의 모든 항목 완료
- 빌드/테스트 통과
- 액션가면의 리뷰 승인
