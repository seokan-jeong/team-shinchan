# Ultrawork Skill

최대 성능 모드를 활성화하여 병렬 에이전트 오케스트레이션으로 작업을 빠르게 완료합니다.

## 트리거

- "ulw", "ultrawork"
- "병렬", "빠르게", "parallel"

## 활성화 시 동작

1. **병렬 실행**: 독립적인 작업을 여러 에이전트가 동시에 처리
2. **적극적 위임**: 전문 에이전트에게 즉시 작업 라우팅
3. **백그라운드 실행**: 긴 작업은 `run_in_background: true`로 실행
4. **완료 보장**: 모든 작업이 검증될 때까지 멈추지 않음

## 검증 체크리스트

작업 완료 전 확인:
- [ ] TODO 리스트: 남은 작업 없음
- [ ] 기능: 요청된 모든 기능 동작
- [ ] 테스트: 모든 테스트 통과
